0.14.0.0 (2025-05-23)

- Added diagonal movement to colonists and monsters
- Added variable movement speed & cost to colonists and monsters; they'll prefer to walk over some blocks over others
-- Briefly; natural terrain blocks are a bit slower, walking through beds, water and such is quite slow, and walking on most player made blocks is a bit faster.
- Doors now block monsters for a bit and discourage them from passing through them
- Added placement previews for torches, rails, flowerboxes, street lanterns, signs, dynamic quarterblocks
- The 'old' flat colored quarterblocks (grey, light & dark brown) are now placable using the same system as the dynamic ones
-- This is a new item; old quarterblocks in the stockpile will be automatically converted on load, and the crafting recipe is also adjusted to make the new one.

- Rails, merchanthub are now removed if the supporting block below them is removed
- Grass can now spread under the berry bushes placed by colonists
- Add colliders to the shaped charge and regular bomb
- Updated kiln/bloomery icons to match new meshes
- Quarterblocks placed on non-ground surfaces now block NPCs
- Prevent placing shaped charges and such inside the player
- Removed on-hit particles from crates
- Smoothed out the "ticks" of block destruction a bit

- Use locale-specific open/close quote tags for item descriptions
- Put the closing quote tag before the period if there is one
- Use locale-specific number separators in the top left/right data
- Fix input icon size in japanese quickstart text

- Somewhat improved moving-target-following logic of monsters
- Fix a sort of memory leak where some pathfinding data was hanging around for previously loaded chunks
- Fix a thread contention issue slowing down autosaving of chunks
- Fix some inconsistencies with lighting objects not directly next to ground (like gliders)
- Mill now correctly has wood particles on destructing
- Fix a bug causing ragdoll instability & clipping into the ground
- Fix banner safe area rendering issue when the player has more than 16 outposts
- Fix a server crash when removing very large stacks of items that are bulk removed when the bottom is removed
- Make it harder to accidentally clip into the ground by placing various blocks (but others are still as bad as they were)
- Fix destruction particles appearing besides the mill on removal
- Fix some text alignment in the outpost creation menu
- Fix label for the "dont show this again" button in the quickstart being too short for some languages
